# Tracking with IMU 

### Installation Steps for Ubuntu

1. Install PCL 
```
$ sudo apt install libpcl-dev
```
2. Run the `setup.sh` script
```
$ ./setup.sh
```
3. To run tracking 
```
$ cd build 
$ ./tracking
```

4. To clean up the output of the tracking, run the `clean.sh` script
```
$ ./clean.sh
```
